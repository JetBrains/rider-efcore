## Commands support roadmap
### Migrations
- [x] Add migration (`dotnet ef migrations add`)
- [x] Remove last migration (`dotnet ef migrations remove`)

### Database
- [x] Update database (`dotnet ef database update`)
- [ ] Drop database (`dotnet ef database drop`)
